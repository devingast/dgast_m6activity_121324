M6 Course Project Documentation

1. About the Project

The M6 Course Project is a simple personal website developed for a Mobile Application Course with Professor Tanner. It's designed to show off web development skills for each Module. The site has a homepage, a favorites page, and a couple of extra pages with more details about specific content. It uses jQuery to add some cool interactions to the site.

2. Website Structure (Site Map)

Here's a quick overview of the pages on the website:

  • Homepage (homepage.html): The first page you see when you visit the site. It has an introduction and links to other pages.
  • Favorites (favorites.html): This page shows your favorite items or content while integrating grids and such.
  • Font Page (fontPage.html): A page showing different font styles and a form.
  • Page1, Page2, and Page3: Testing pages for swatches and theme roller.

3. Actions and Buttons

Here's a quick rundown of what happens when you click things:

  • Homepage (homepage.html):
    • Navigation Button: Takes you to pages like "Favorites" and "Page 1".
    • Interactive Section: Click to see more details about items on the homepage.
  • Favorites Page (favorites.html):
    • Add to Favorites Button: Lets you save items to your favorites list.
    • Remove Button: Lets you remove items from your favorites.

4. Where to Find the Site

5. Code Breakdown

HTML Files:

  • homepage.html: The main page where users start.
  • favorites.html: Page where users can view and manage favorites.
  • fontPage.html: A page with font examples and related content.
  • page1.html, page2.html, page3.html: Other pages with different info.

JavaScript:

  • jquery-3.7.1.min.js: A library that powers interactive features on the site, like smooth scrolling and interactive elements.

Themes Folder:

  • Contains CSS files that define the look and feel of the site, ensuring a consistent design across all pages.
